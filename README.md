# Information Aggregator

一个现代化的信息聚合平台，帮助您高效管理和发现有价值的内容。

## 功能特性

- **日报视图** - 每日精选内容概览
- **周报视图** - 本周热点和深度分析
- **收藏功能** - 保存感兴趣的文章

## 快速开始

### 环境要求

- Node.js 20.19+ 或 22.12+
- pnpm 9+

### 安装

```bash
pnpm install
cp .env.example .env
# 编辑 .env 填入数据库连接信息
```

### 开发

```bash
pnpm dev
```

## 技术栈

- [Next.js 16](https://nextjs.org/)

- [shadcn/ui](https://ui.shadcn.com/)
