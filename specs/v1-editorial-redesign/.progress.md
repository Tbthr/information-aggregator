# Progress: v1-editorial-redesign

## Original Goal

将信息聚合器从"聚合工具"升级为"个人注意力管理系统"，包含两个核心方向：

### 1. 产品策略层（PRD 1）
- 建立统一来源策略框架（Source Policy / Content Treatment Policy）
- X 场景深化（x_bookmarks, x_home, x_list 等差异化处理）
- 日报作为 Web 主入口
- AI 判断与解释链路（keepDecision, keepReason, readerBenefit 等）

### 2. Web UI 重设计（PRD 2）
- 编辑部式视觉系统（现代杂志 + 研究简报风格）
- 日报首页：封面故事 + Lead Story + Top Signals + Scan Brief
- Pack 视图：策展单元，而非简单过滤列表
- 来源视图：展示策略模式（assist_only / filter_then_assist）
- 周报视图：编辑部回顾，主题聚合

### 核心原则
- Pipeline 保持确定性，AI 作为可选增强层
- 首屏不暴露工程参数，Pack/来源为一级切换维度
- 卡片层可见推荐解释，不藏在详情页
- 视觉气质：强调阅读可信度、层级和节奏感
