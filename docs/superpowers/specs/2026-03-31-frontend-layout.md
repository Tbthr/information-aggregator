# 前端布局设计

## Aesthetic Direction: Editorial/Magazine

**灵感：** 老派报纸排版 + 现代数字阅读体验

### 视觉风格

| 元素 | 选择 | 理由 |
|------|------|------|
| 标题字体 | Newsreader (serif) | 新闻感、编辑感 |
| 正文字体 | IBM Plex Sans | 技术内容好读 |
| 主色调 | #1a1a1a (墨黑) | 纸墨质感 |
| 背景色 | #faf8f5 (暖白) | 舒适阅读 |
| 强调色 | #c45d3a (赤陶) | 温暖但不艳 |

### 布局结构

```
┌─────────────────────────────────────────────────────────┐
│  THE DAILY BRIEF                           [日] [周]    │  ← 顶部导航
├──────────────┬──────────────────────────────────────────┤
│              │                                          │
│   DATE       │   # 今日主题分类                         │
│   ─────      │                                          │
│   03-31      │   摘要内容...                            │
│   03-30      │                                          │
│   03-29      │   ## 精选                                │
│   ...        │                                          │
│              │   - 文章1 (reason)                       │
│   WEEK       │   - 文章2 (reason)                       │
│   ─────      │                                          │
│   W13        │                                          │
│   W12        │                                          │
│   W11        │                                          │
│              │                                          │
└──────────────┴──────────────────────────────────────────┘
```

### 动效

- 页面切换：淡入淡出 (300ms ease)
- 列表 hover：轻微右移 + 强调色下划线
- 内容加载： staggered 渐显

### 技术实现

- 单 HTML 文件 + vanilla CSS/JS
- markdown-it 浏览器端渲染
- 无需构建工具
