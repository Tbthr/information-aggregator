• Anthropic 发布最新经济研究：追踪 Claude Code 规模化使用数据 — 谁在用、用来做什么、任务价值如何变化、开发者产出提升多少。首次系统性量化 AI coding agent 的经济影响 — feed

• Blueberry：首个 commerce agent，给目标后自动找到互动用户、调研并卖货。评论者认为这是「社会工程学的 agent 化」，在中国隐私环境下会极高效 — feed

• Slack 支持 HTML 附件渲染：不再把 HTML 显示为恐怖的文本 blob，开发者社区热议 — feed

• SEC 注册 AI 投资顾问：某项目将 AI agent 本身注册为 SEC 投资顾问，用户可用自然语言管理投资组合，合规 + AI agent 的前沿交叉 — feed
