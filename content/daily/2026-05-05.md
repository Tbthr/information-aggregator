## AI快讯

### 产品更新

- [**豆包拟推三档订阅服务，月费最高五百元**](https://apps.apple.com/cn/app/%E8%B1%86%E5%8C%85-%E9%9A%8F%E6%97%B6%E5%B8%AE%E5%BF%99%E7%9A%84-ai-%E5%8A%A9%E6%89%8B/id6459478672) — 豆包 App Store 页面近日被发现有每月 68元 到 500元 不等的三档付费服务定价。
据媒体报道，豆包 官方回应称产品将始终提供免费服务，目前正探索更多增值服务，相关方案仍处测试阶段。
- [**TinyFish 宣布搜索与抓取功能免费面向开发者开放**](https://www.tinyfish.ai/blog/search-and-fetch-are-now-free-for-every-agent-everywhere) — TinyFish宣布，其Web Search与Fetch两项核心功能即日起面向所有开发者和 Agent 免费开放。
免费套餐包含每分钟 5 次搜索查询和每分钟 25 次 URL 抓取的速率限制。
- [**Higgsfield 上线 CLI 技能支持 30 余种模型生成**](https://higgsfield.ai/cli) — Higgsfield宣布推出CLI与Marketing Skills。现在，任何兼容MCP的Agent都能直接调用三十多种模型。
这些模型能生成最高 4K 图像与最长 15秒 视频。
- [**Google 为 Gemini API 引入事件驱动 Webhooks 推送通知功能**](https://blog.google/innovation-and-ai/technology/developers-tools/event-driven-webhooks) — Gemini API 引入了 Event-Driven Webhooks 功能。
面对 Agent 工作流等长耗时任务，API 会在完成时主动向服务器推送 HTTP POST 通知，告别低效轮询。
- [**Claude API平台推出 Workload Identity Federation功能**](https://platform.claude.com/docs/en/build-with-claude/workload-identity-federation) — Anthropic 旗下的 Claude API 平台推出了 Workload Identity Federation 功能。
开发者可利用 AWS、GCP 或 GitHub Actions 等环境的短时效 JWT 令牌进行无密钥认证调用 API。

### 开源项目

- [**Vercel 开源 deepsec 基于 AI Agent 的代码安全扫描框架**](https://vercel.com/blog/introducing-deepsec-find-and-fix-vulnerabilities-in-your-code-base) — Vercel 开源了基于 coding Agent 的安全扫描框架 deepsec。
该工具能在本地直接运行，通过五步工作流精准发现大规模代码库中的隐蔽漏洞。
- [**Cursor 团队发布开源内部工作流插件 Cursor Team Kit**](https://cursor.com/cn/marketplace/cursor/cursor-team-kit) — Cursor团队发布了其内部工作流插件Cursor Team Kit。
该套件内置 17个 Skills、1 个Agent与2条规则，全面打包了在代码审查、测试验证及自动修复等环节的自动化实践。
- [**OpenClaw 发布 2026.5.3 版本，新增文件传输与实时控制功能**](https://github.com/openclaw/openclaw/releases/tag/v2026.5.3) — OpenClaw 发布了 2026.5.3 版本。
新版本支持配对节点间文件传输，同时引入了 /steer 命令与 /side 命令，分别能实时干预任务以及进行不污染上下文的快速提问。
此外，插件体系全面转向官方 npm 包，系统性能与稳定性也得到显著提升。

### 行业动态

- [**Anthropic 联合黑石成立新公司，助 Claude 部署**](https://www.anthropic.com/news/enterprise-ai-services-company) — Anthropic宣布与Blackstone等投资方联合成立一家AI服务公司，并加入Claude Partner Network。
通过专属工程团队帮助中型企业，将**Claude**模型定制部署到其核心业务中。
- [**Cerebras 启动 IPO 路演，估值 266 亿美元**](https://www.cerebras.ai/press-release/cerebras-systems-announces-launch-of-initial-public-offering) — Cerebras Systems 正推进首次公开募股。
该公司计划最高募资 35亿美元。
此轮募资对应市值约为 266.2亿美元。
- [**传特朗普政府考虑组建工作组在 AI 模型发布前实施安全审查**](https://www.nytimes.com/2026/05/04/technology/trump-ai-models.html) — 据报道，受Anthropic推出的Mythos模型所触发，白宫已向科技企业通报。
正考虑发布行政命令成立政企联合的AI工作组，以研究在AI模型公开发布前实施政府安全审查的流程。
