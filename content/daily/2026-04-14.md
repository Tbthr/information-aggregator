## AI快讯

### 产品更新

- [**月之暗面上线 Kimi k2.6-code-preview 模型**](https://x.com/Hx1u0/status/2043738521920516129) — 月之暗面宣布旗下 Kimi Code 的抢先体验计划已结束。根据官网信息，参与测试的新模型 k2.6-code-preview 已上线。与此同时，该团队已修复因 Claude Code 相关版本导致模型表现下降的问题。
- [**OpenRouter 上线新 stealth 模型 Elephant Alpha**](https://x.com/OpenRouter/status/2043714975756390844) — OpenRouter 宣布上线一款代号为 Elephant Alpha 的千亿参数 stealth 文本模型，具备 256K 上下文窗口与 32K 最大输出。目前用户可免费接入该模型，但需注意提供商会记录所有提示词与生成内容。
- [**Google AI Studio 引入 Tab Tab Tab 引擎优化提示词补全体验**](https://x.com/miketaylorcai/status/204375326) — Google AI Studio 上线了名为 Tab Tab Tab 的全新提示词自动补全引擎。开发者在编写时只需输入初步且模糊的想法，该引擎就能自动为你填补内容空白并完善提示词的具体细节。
- [**Qwen 下调 OAuth 免费配额至百次服务即将停服**](https://github.com/QwenLM/qwen-code) — Qwen Code 官方宣布，将 Qwen OAuth 免费层的每日请求配额从 1000 次下调至 100 次，且该免费服务将在 4月15日 彻底停止。

### 开源项目

- [**OpenClaw 发布 2026.4.12 版，新增本地模型与安全修复**](https://github.com/openclaw/openclaw/releases/tag/v2026.4.12) — OpenClaw 官方发布 2026.4.12 版本，该版本内置 LM Studio provider，强化本地模型支持，并为 macOS Talk Mode 引入实验性本地 MLX speech provider。同时还集中修复了大量涉及安全、网关及多平台通信组件的缺陷。

### 前沿研究

- [**英国 AISI 报告称 Claude Mythos 可完成 32 步网络攻击**](https://www.aisi.gov.uk/blog/our-evaluation-of-claude-mythos-previews-cyber-capabilities) — 英国AI安全研究所的最新评估报告指出，Anthropic推出的Claude Mythos Preview模型展现出前所未有的网络攻击能力。它不仅以极高的成功率攻克了专家级网络安全挑战，更成为首个能够自主完成包含数十个步骤的企业网络攻击全流程的AI模型。

### 行业动态

- [**Cloudflare 联合 OpenAI 将 GPT-5.4 及 Codex 引入 Agent Cloud**](https://openai.com/index/cloudflare-openai-agent-cloud/) — Cloudflare 与 OpenAI 宣布达成合作，正式将 GPT-5.4 等模型引入 Cloudflare Agent Cloud 平台。同时，Codex harness 已在 Cloudflare Sandboxes 安全虚拟环境中正式商用。
- [**软银等四日企联合成立公司，将开发万亿参数 AI 模型**](https://www.nikkei.com/article/DGXZQOUC1207B0S6A410C2000000/) — 据报道，软银、NEC、索尼集团与本田在东京共同出资成立新公司“日本AI基础模型开发”，计划在未来数年内构建参数达 1万亿 的日本最大规模多模态基础模型。
- [**MiniMax 回应 M2.7 模型许可证变更争议**](https://x.com/RyanLeeMiniMax/status/2043573044065820673) — MiniMax 工作人员回应 M2.7 模型许可证变更争议。由于此前多次出现第三方托管服务提供劣质甚至虚假的模型接口，严重损害了该公司的声誉和用户体验，因此决定对该模型的商业服务实施授权限制。
- [**Anthropic 回应提示词缓存时长争议**](https://x.com/bcherny/status/2043715713551212834) — 针对开发者质疑 Claude Code 在关闭遥测后，提示词缓存时长会从一小时降至五分钟的争议，官方工程师澄清，这并非牺牲隐私换性能的惩罚机制，而是客户端实验开关失效导致读取了默认值。
并称未来更新中将提供环境变量，允许所有用户根据实际需求手动强制切换缓存时长。
- [**Cursor 回应内置 Claude Agent SDK 争议**](https://x.com/dotey/status/2043588734952563180) — 有开发者逆向发现，Cursor 3.0 不仅内置了完整的 Claude Agent SDK，还将相关标识全面替换为 Cursor。
Cursor 创始人对此澄清，这是在不到 1% 流量上进行的 A/B 测试，目的是对比 Claude 框架与 Cursor 框架的表现差异。
- [**OpenAI CEO Sam Altman 住宅再遭袭击**](https://www.sanfranciscopolice.org/news/sfpd-arrests-suspects-involved-shooting-26-044) — OpenAI CEO Sam Altman 位于旧金山的住宅再次遭遇袭击事件。当地时间周日，有嫌疑人向其住宅开枪射击，涉事的嫌疑人均已被警方逮捕。
