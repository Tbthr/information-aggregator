## AI快讯

### 产品更新

- [**DeepSeek 发布 DeepSeek-V4 系列模型**](https://api-docs.deepseek.com/zh-cn/news/news260424) — DeepSeek正式发布并开源了全新的DeepSeek-V4系列预览版模型，包含 Pro 和 Flash 两个版本，均支持 100万 超长上下文。
官方称，DeepSeek-V4-Pro 性能比肩顶级闭源模型，Agent 能力大幅提升，拥有丰富的世界知识和世界顶级的推理性能。
而 DeepSeek-V4-Flash 则是更快捷高效的经济之选。两款模型已在官网、App 以及官方 API 中上线。
- [**OpenAI 已在 API 中上线 GPT-5.5 和 GPT-5.5 Pro**](https://developers.openai.com/api/docs/guides/latest-model) — OpenAI 宣布，GPT-5.5 和 GPT-5.5 Pro 已在 API 中上线。
部分第三方平台已宣布接入该系列模型。
- [**阶跃星辰发布 StepAudio 2.5 ASR 语音识别模型**](https://mp.weixin.qq.com/s/jGeJxEsGHaLNT00BoqcTZw) — 阶跃星辰发布了语音识别模型 StepAudio 2.5 ASR。官方称该模型能一次性端到端完整处理 30 分钟 的音频，中英双语识别精度均达到业内顶尖水平。
该模型已上线 Step Plan，并开放了体验中心及相关 Demo 供用户体验。
- [**Claude Code 更新 Web 端和移动端界面，新增跨平台无缝切换功能**](http://claude.ai/code) — Claude 宣布升级 Claude Code 的网页端体验，统一视觉设计并重建底层架构，还推出了跨平台无缝切换功能。
用户只需运行 claude --teleport 命令，就能在网页、手机和终端之间保留完整记录继续工作。
- [**NotebookLM 推出来源自动分类**](https://x.com/NotebookLM/status/2047735817003499790) — NotebookLM 官方宣布推出来源自动打标签与分类功能。
当笔记中添加了 5 个 及以上来源时，系统会自动进行分类整理。
这一新功能旨在解决笔记管理难题。

### 开源项目

- [**Hermes Agent 发布 v0.11.0，支持多个新模型与 QQBot**](https://github.com/NousResearch/hermes-agent/releases/tag/v2026.4.23) — NousResearch团队正式发布了Hermes Agent v0.11.0版本。
新版本全面重构了终端交互界面，接入了包括GPT-5.5在内的多个新模型以及QQBot。
同时大幅增强了插件扩展和子Agent协调能力。
- [**OpenClaw 发布新版支持 GPT-5.5 模型并强化安全防护**](https://github.com/openclaw/openclaw/releases/tag/v2026.4.23) — OpenClaw 发布了 2026.4.23 版本，本次更新重点强化了系统底层架构与多平台通道适配。
新增了 分支上下文子Agent 以提升会话灵活性，还引入 Codex OAuth 进行图像生成的功能。
- [**ComfyUI 完成 3000 万美元融资，估值达 5 亿美元**](https://blog.comfy.org/p/comfyui-raises-30m-to-scale-open) — ComfyUI宣布完成3000万美元融资，估值达到5亿美元。
该平台计划利用这笔新资金扩展云服务与协作功能。
同时郑重承诺该产品将永远保持开源。

### 前沿研究

- [**Anthropic 发布名为 Project Deal 的实验研究结果**](https://www.anthropic.com/features/project-deal) — Anthropic开展了一项名为Project Deal的实验，让 Claude代替员工在虚拟市场中自主进行物品交易和讨价还价。
实验发现，AI 能独立完成上百笔真实交易。使用高级模型 Opus的 AI 在成交价和成交量上占据明显优势。

### 行业动态

- [**Meta 部署数千万 AWS Graviton5 核心，支持 agentic AI 计算**](https://about.fb.com/news/2026/04/meta-partners-with-aws-on-graviton-chips-to-power-agentic-ai/) — Meta宣布与 AWS 达成协议，计划将数千万个 AWS Graviton5 处理核心纳入计算资源组合。
此举旨在支持复杂的 agentic AI 工作负载并推进基础设施多元化战略。
- [**Cohere 联手 Aleph Alpha 组建主权 AI，Schwarz 领投 6 亿美元**](https://cohere.com/blog/cohere-alephalpha-join-forces) — 加拿大 AI 企业 Cohere 与 德国 AI 企业 Aleph Alpha 宣布联手组建一家全球性 AI 巨头。
这一合作将汇集双方的人才与算力资源，共同致力于构建安全可控的主权 AI 解决方案。
该方案主要面向高度监管行业，旨在满足其对数据主权与技术合规的严苛要求。
- [**传谷歌计划向 Anthropic 投资最高 400 亿美元**](https://techcrunch.com/2026/04/24/google-to-invest-up-to-40b-in-anthropic-in-cash-and-compute/) — 据报道，谷歌计划向Anthropic投资最高400亿美元。
该投资包含先期100亿美元现金和后续的业绩达标追加投资。
同时，Google Cloud将在未来五年内为其提供全新的5吉瓦计算容量，以满足Anthropic日益庞大的算力需求。
