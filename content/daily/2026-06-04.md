## AI快讯

### 产品更新

- [**豆包计划推出专业版 辟谣降体验推会员传闻**](https://mp.weixin.qq.com/s/Be5hOIJUQC8ogxRy3epWLQ) — 豆包官方宣布计划推出“豆包专业版”，目前尚处测试阶段。官方承诺日常基础功能保持免费，并明确辟谣网传“降体验逼买会员”的不实说法。
- [**Reve 发布 Reve 2.0 图像模型**](https://x.com/reve/status/2062260665121919101) — AI 图像实验室 Reve 正式发布 Reve 2.0 图像模型。官方宣称该模型是世界最佳的 4K 图像模型，可通过精确布局控制图像生成与编辑，目前已于 今天 上线可用。
- [**OpenAI 升级生命科学模型 GPT-Rosalind**](https://openai.com/index/introducing-new-capabilities-to-gpt-rosalind/) — OpenAI 为专用于企业级生命科学研究的 GPT-Rosalind 推出大升级，整合了 GPT-5.5 的 agentic coding 与工具使用能力。官方称其在多项基准测试中表现领先，现正向全球符合资格的机构开放研究预览版。
- [**OpenAI 调整 Codex 验证规则，服务端现 429 故障**](https://status.openai.com) — 据社区反馈，OpenAI 疑似取消 Codex 强制手机二次验证，但仍有部分用户称需要验证，同时系统疑似灰度测试 Passkey 与邮件接码。还有用户观察到，有人密集在 OpenAI 官方论坛发布了关于跨区订阅等的举报。此外，北京时间 6月3日，OpenAI 服务端突发大规模故障，Codex 与 API 频繁出现报错，Codex 额度显示异常。
- [**Codex 宣布重置所有付费套餐使用额度**](https://x.com/thsottiaux/status/2062329981548802523) — 针对过去24小时内多起影响 Codex 可靠性的事件，Codex 负责人 Tibo 于今日上午约8点半宣布重置所有付费套餐使用额度。
- [**MiniMax 优化 M3 模型速度并开启 Token Plan 退款**](https://platform.minimaxi.com/docs/token-plan/migration) — MiniMax宣布本周五上午十点前购买 Token Plan 的用户，将获M3周限额永久额外百分之五十的福利。官方上线专属自助售后平台接管答疑，控制台同步开放Token Plan退款通道，常规订单本周内处理完毕。
- [**OpenClaw 更新：新增 Skill Workshop 与原生 Windows 支持**](https://openclaw.ai/blog/openclaw-agent-skill-workshop) — OpenClaw 发布 2026.6.1 版本，新增 Skill Workshop 允许 Agent 将重复工作转化为可审查的复用技能，并正式引入原生 Windows 节点支持。
- [**讯飞上线星辰Token Plan包月订阅服务**](https://mp.weixin.qq.com/s/CBD4Vak-sAR_UWZN_APBEw) — 讯飞官方推出星辰Token Plan包月订阅服务，官方称支持多款旗舰模型与讯飞核心能力统一调用，提供高峰不限流及最高500万TPM吞吐保障。
- [**Kimi 推出本地 Agent 产品 Kimi Work Beta 版**](https://mp.weixin.qq.com/s/guE7g4kpu_PdrDT0a7DhzA) — Kimi 官方推出本地 Agent 产品 Kimi Work Beta 版。该产品内核基于 Kimi Code，最高支持 300 个子 Agent 并行协作，目前已开放 Mac 版下载，Windows 版即将上线。
- [**千问向第三方 Agent 和 Skill 全面开放并开启首批测试**](https://mp.weixin.qq.com/s/rUcQ-QxC3xmKfrK18fV0Yw) — 千问宣布向第三方 Agent 和 Skill 全面开放。瑞幸咖啡、肯德基等首批企业正在测试并陆续上线，未来所有企业均可运营自定义品牌 Agent。
- [**QoderWork 升级高校福利：师生完成认证额外获 4000 积分**](https://mp.weixin.qq.com/s/EsEY_OWq70U0dcHQjnXokw) — QoderWork 中国版面向高校师生推出专属福利，完成身份认证后可额外获得有效性3 个月的4000积分，累计获得6000 积分。
- [**Google Labs 发布 Dreambeans：利用 Google 数据生成生活故事**](https://blog.google/innovation-and-ai/models-and-research/google-labs/dreambeans/) — Google Labs 推出实验性应用 Dreambeans，基于用户 Google 数据生成个性化“故事”，目前仅限符合条件的美国 Google AI Ultra 订阅用户使用。
- [**Google Search 允许网站不被 AI Overviews 收录**](https://blog.google/products-and-platforms/products/search/new-controls-website-owners/) — 据报道，应 英国 反垄断机构要求，Google 在 Search Console 中测试新控件，允许 英国 网站屏蔽其内容被生成式 AI 搜索使用，后续将向全球推广。
- [**OpenAI 计划将 Codex 核心能力整合进 ChatGPT**](https://openai.com/zh-Hans-CN/business/intelligence-at-work/) — OpenAI 在近期的产品发布直播中提及将在未来几周把 Codex 核心能力整合进 ChatGPT，用户无需切换产品即可在多平台调用 Agent，ChatGPT 将升级为统一工作界面。

### 开源项目

- [**Google 发布 Gemma 4 12B 原生多模态模型**](https://blog.google/innovation-and-ai/technology/developers-tools/introducing-gemma-4-12b/) — Google DeepMind 发布 Gemma 4 12B 模型。它采用无编码器架构支持原生音视频输入，可在 16GB 内存笔记本本地运行多模态 Agent，现已基于 Apache 2.0 协议开源。
- [**Ideogram 发布开源图像模型 Ideogram 4.0**](https://ideogram.ai/blog/ideogram-4.0/) — Ideogram 发布开源图像生成模型 Ideogram 4.0。该模型参数量为9.3B，支持结构化 JSON 提示词和边界框布局控制，官方称其在多语言文本渲染和设计能力上达到业界领先水平，并在多个第三方基准测试中位居开源模型首位。
- [**Nex AGI 发布并开源 Nex-N2-Pro 模型**](https://huggingface.co/nex-agi/Nex-N2-Pro) — Nex AGI 发布并开源模型 Nex-N2-Pro。官方称其具备自适应推理能力，性能媲美前沿模型。该模型已在硅基流动平台提供限时免费调用。
- [**京东开源 JoyAI-Echo 长音视频生成框架**](https://github.com/jd-opensource/JoyAI-Echo) — 京东 Joy Future Academy 发布 JoyAI-Echo 框架，支持生成长达 5 分钟的连贯音视频，代码与权重已开源。
