### 产品更新

- [**DeepSeek 调整 API 计费，输入缓存命中价降至首发价十分之一**](https://api-docs.deepseek.com/zh-cn/quick_start/pricing) — DeepSeek官方宣布，即日起将全系列模型API服务的输入缓存命中价格统一降至首发价格的十分之一，永久生效。
同时，目前V4-Pro模型的输入缓存命中价在降价的基础上，叠加了此前公布的限时2.5折优惠。
- [**Qoder 开启限时优惠 Ultimate 档位模型计费倍率降至 0.8x**](https://x.com/qoder_ai_ide/status/2048415481346834551) — Qoder 宣布开启限时优惠，Ultimate档位模型的计费倍率直接减半降至 0.8x。
这项限时优惠会在 IDE、JetBrains插件及 CLI 中自动生效。
- [**AWS 将推出 Claude Platform on AWS 服务整合原生体验**](https://aws.amazon.com/claude-platform/) — Amazon Web Services 官方发布预告，即将推出 Claude Platform on AWS 服务。
这项服务能让开发者直接利用现有的 AWS 账户体系与安全机制。
用户将能够无缝体验 Anthropic 原生的 Claude 控制台和全量功能。
