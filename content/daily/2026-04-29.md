## AI快讯

### 其他

- [**Claude 推出创意软件连接器，支持 Blender 和 Adobe 系列等多种工具**](https://www.anthropic.com/news/claude-for-creative-work) — Claude 推出了全新的创意软件连接器，用户现在可以直接通过对话界面无缝操控 Blender 和 Adobe 系列等多款行业标准工具。
借助这一功能，用户可以轻松完成 3D 建模和音视频编辑等创意工作，极大提升了创作效率。
- [**DeepSeek 延长 V4-Pro 模型 2.5 折优惠至 5 月 31 日**](https://api-docs.deepseek.com/zh-cn/quick_start/pricing) — DeepSeek宣布将DeepSeek-V4-Pro模型的2.5折优惠期，延长到 2026年5月31日 23:59。
在这个时间前调用该模型，都可以享受2.5折的特惠价格。
- [**阶跃星辰上线智能路由及API限免图像编辑模型**](https://platform.stepfun.com/docs/zh/api-reference/images/image) — 阶跃星辰开放平台宣布近期上线了多项更新。
主要包括结合 DeepSeek V4 Pro 组合的 Step Plan 智能路由方案。
同时还有 API 限时免费的图像编辑模型等。
- [**蚂蚁集团开源 Ling-2.6-flash 模型，总参数 104B**](https://huggingface.co/inclusionAI/Ling-2.6-flash) — 蚂蚁百灵正式开源了 Ling-2.6-flash 大模型。
这是一款总参数量达 104B、激活参数量为 7.4B 的非推理模型。
目前，该模型权重已在相关平台全面开放下载。
- [**英伟达发布 Nemotron 3 Nano Omni 开源多模态模型**](https://developer.nvidia.com/blog/nvidia-nemotron-3-nano-omni-powers-multimodal-agent-reasoning-in-a-single-efficient-open-model/) — 英伟达发布了开源多模态模型 Nemotron 3 Nano Omni。
该模型能在单一推理循环中统一处理视频、音频、图像和文本，大幅降低推理成本。
目前其模型权重和训练方法已全面开源。
- [**商汤科技开源 SenseNova U1 原生多模态模型**](https://huggingface.co/sensenova/SenseNova-U1-8B-MoT) — 商汤发布了原生多模态模型系列 SenseNova U1，并开源了包含两个规格的 Lite 版本。
该模型在单一架构中统一了视觉理解、推理与图像生成能力。
同时，它支持连贯的图文交错生成。
- [**Poolside 发布 Laguna 系列模型，XS.2 开源权重**](https://poolside.ai/blog/laguna-a-deeper-dive) — Poolside发布了Laguna系列两款MoE模型，包括Laguna M.1和专为本地编码设计的开源模型Laguna XS.2。
两款模型目前已限时免费开放使用。
- [**研究团队开源 130 亿参数复古模型，知识截止 1930 年**](https://huggingface.co/talkie-lm) — 一支研究团队开源了拥有130亿参数的复古语言模型 Talkie-1930。
这款模型的知识被严格冻结在 1930年 的年末，团队完全使用 1931年 之前的历史文本进行训练。
- [**OpenAI 确认本周将有更新，重置 Codex 速率限制**](https://x.com/thsottiaux/status/2048997818673537399) — OpenAI 确认本周将继续发布更新。
同时，Codex 负责人于北京时间 4月28日 下午宣布，为所有付费用户重置 Codex 的速率限制，方便大家使用 GPT-5.5 进行开发。
- [**Warp 开源 AI 终端客户端代码，服务端保持闭源**](https://www.warp.dev/blog/warp-is-now-open-source) — Warp 正式在 GitHub 开源了用 Rust 编写的终端客户端代码，但服务端仍保持闭源。
这次开源引入了全新的协作模式，让人类专注于提出创意和验证结果，而代码编写和测试等繁重工作则交由云端 Agent 来完成。
同时官方还为 Warp 新增了多款开源模型的支持和界面定制功能。
- [**OpenClaw 发布 2026.4.26 版本，深度重构插件系统**](https://github.com/openclaw/openclaw/releases/tag/v2026.4.26) — OpenClaw 发布了 2026.4.26 版本，这次更新对插件系统进行了深度重构。
让各项启动和响应时间大幅缩短至毫秒级。
新版本还引入了 Google Live Talk 模式和全新迁移命令。
并全面修复了大量本地模型问题。
- [**支付宝正式上线 AI 收，个人开发者至 2026 年享零费率**](https://mp.weixin.qq.com/s/21BdnLtGqXB5plPpoU9hUg) — 支付宝正式上线了AI 收服务，提供AI 服务的商家和个人开发者只需简单接入。

- [**LiteLLM 修复 SQL 注入漏洞，未授权可读取密钥**](https://mp.weixin.qq.com/s/ytNWdqGECo0fmWwPQGqy8A) — LiteLLM 近期修复了一个高危的 SQL 注入漏洞，攻击者可能借此窃取数据库中保存的 API 密钥。
