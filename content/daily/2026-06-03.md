## AI快讯

### 产品更新

- [**OpenAI 为 Codex 推出六款业务插件与 Sites 和 Annotations功能**](https://openai.com/index/codex-for-every-role-tool-workflow/) — OpenAI为Codex推出多项新功能，将应用拓展至更广泛的知识工作场景。此次更新包含数据分析等六类专属业务插件，可将内容转化为交互式网站的Sites功能，以及支持局部精确修改的Annotations功能。这些更新正面向Business和Enterprise用户推出预览版。
- [**Windsurf 升级更名为 Devin Desktop，支持 ACP**](https://devin.ai/blog/windsurf-is-now-devin-desktop/) — Cognition 宣布将 Windsurf 升级为 Devin Desktop，将其定位为统一管理本地与云端多 Agent 的控制中心。该产品现已开放下载，支持通过 ACP 协议接入 Codex等第三方 Agent，现有用户将自动更新且维持原定价。
- [**小米 MiMo 推出老用户专属福利**](https://platform.xiaomimimo.com/console/plan-manage) — 小米MiMo发布了Token Plan老用户专属福利。5月27日0点前过期的付费用户，将免费获赠相同的Token Plan，无需操作直接到账。5月27日0点后续费的用户，将获得等额礼金用于抵扣API消耗费用。以上权益的有效期均为一个自然月。
- [**Antigravity 更新 Gemini 3.5 Flash 并推出 CLI 会话同步**](https://x.com/antigravity/status/2061928539499421968) — Antigravity 宣布上线新版 Gemini 3.5 Flash，官方称其废话更少且处理困难任务的持久力更强，同时已为所有用户重置了速率限制。同时， Antigravity CLI 更新至 v1.0.4，支持从 Antigravity 2.0 同步对话。
- [**Claude Code 新版 /fork 支持携带完整上下文进行后台处理**](https://x.com/ClaudeDevs/status/2061947411141169494) — ClaudeDevs 宣布更新 Claude Code 的 /fork 命令，使其能在后台运行携带完整上下文的 Agent 并返回结果，而原功能被重命名为 /branch。
- [**Anthropic 发布 ant 命令行工具覆盖 Claude Platform API 全端点**](https://platform.claude.com/docs/en/api/sdks/cli) — Anthropic 发布 ant 命令行工具，支持在终端直接调用 Claude Platform 全部 API 端点，涵盖 Messages API 与 Managed Agents，Claude Code 原生集成。
- [**GitHub 宣布多项 Copilot 更新，App 预览版扩大范围且 SDK 正式发布**](https://github.blog/changelog/2026-06-02-expanded-technical-preview-availability-for-the-github-copilot-app) — GitHub 官方宣布 Copilot 系列迎来多项更新：Copilot App 技术预览版已扩大至多层级订阅用户，Copilot SDK 正式发布，同时引入安全沙盒、CLI 实验性终端 UI、可定制代码审查及 Partner Agent apps。
- [**Nous Research 推出 Hermes Desktop 原生桌面应用**](https://hermes-agent.nousresearch.com/desktop) — Nous Research 宣布推出 Hermes Desktop，这款开源 Agent 的原生桌面应用现已进入公开预览版阶段，支持在 macOS、Windows 和 Linux 系统上运行。
- [**Anthropic 向超 15 国扩展 Claude Mythos Preview 访问权限**](https://www.anthropic.com/news/expanding-project-glasswing) — Anthropic 官方宣布扩展 Project Glasswing，将 Claude Mythos Preview 访问权限扩大至约 150 家新组织，用于扫描关键代码库漏洞。
- [**微软发布 7 款 MAI 模型，旗舰模型 MAI-Thinking-1 开启私有预览**](https://microsoft.ai/news/building-a-hillclimbing-machine-launching-seven-new-mai-models/) — Microsoft AI 官方发布 7 款 MAI 模型，其中旗舰推理模型 MAI-Thinking-1 现已在 Microsoft Foundry 开放私有预览。该模型拥有 1T 总参数和 35B 激活参数，官方称其在未使用第三方模型蒸馏的从零训练下，于 SWE-Bench Pro 等基准测试中达到了领先模型的水平。
- [**微软发布 MAI-Image-2.5 模型，Arena 图像编辑排第二**](https://microsoft.ai/news/introducing-mai-image-2-5/) — 微软发布了MAI-Image-2.5及其Flash版本图像模型，主打高保真生成与精准局部编辑，在Arena图像编辑榜位居第二。目前，该系列模型已集成至 PowerPoint，并面向开发者在 Foundry平台开放调用。
- [**微软发布MAI-Voice-2语音模型 支持15种语言及零样本生成**](https://microsoft.ai/news/mai-voice-2expressive-speech-in-10-languages/) — 微软AI正式发布新一代文本转语音模型MAI-Voice-2，现已支持包括简体中文在内的15种语言，并已在Azure Foundry上线且正逐步集成至VSCode等产品中。根据官方提供的数据，其生成的语音已达到与真人录音难以区分的程度。
- [**微软推出 MAI-Transcribe-1.5 语音转文本模型**](https://microsoft.ai/news/mai-transcribe-1-5more-accurate-context-aware-and-built-for-production/) — Microsoft 正式推出多语言语音转文本模型 MAI-Transcribe-1.5，官方称其支持 43 种语言且具备同类最佳的词错率，目前已集成至 Copilot、Teams 等产品并在 Foundry 中可用。
- [**微软推出 MAI-Code-1-Flash 编码模型**](https://microsoft.ai/news/introducingmai-code-1-flash/) — 微软推出编码模型 MAI-Code-1-Flash，现面向 VS Code 的 GitHub Copilot 个人用户陆续推出。官方称，其表现优于 Claude Haiku 4.5。
- [**微软发布 Project Solara 平台，专为 Agent 优先设备打造**](https://commandline.microsoft.com/project-solara-build-2026/) — 微软 发布了 Project Solara，这是一个专为 Agent 优先 设备打造的端云融合平台。该平台基于 Android 开发，目前正以两款概念参考设计进行内部测试并计划展开企业试点。
- [**微软发布 Surface RTX Spark Dev Box 主机**](https://blogs.windows.com/devices/2026/06/02/building-the-next-generation-of-devices-for-developers-surface-rtx-spark-dev-box/) — Microsoft 推出专为本地 AI 开发设计的紧凑型主机 Surface RTX Spark Dev Box。该设备搭载 NVIDIA RTX Spark 超级芯片与 128GB 统一内存，预计今年晚些时候在美国市场独家发售。
- [**微软发布 Majorana 2 量子芯片**](https://aka.ms/m2blog) — 微软发布 Majorana 2 拓扑量子芯片，借力 Agent 将量子比特可靠性提升千倍。同时 Microsoft Discovery 平台全面可用，并将可扩展量子计算机预期提前至 2029年。
- [**微软发布 MXC SDK 早期预览版，为 AI Agent 提供沙箱隔离**](https://blogs.windows.com/windowsdeveloper/2026/06/02/windows-platform-security-for-ai-agents/) — 微软发布了 Microsoft eXecution Containers (MXC) SDK 的早期预览版，为 Windows 等平台上的 AI Agent 提供跨平台、策略驱动的沙箱隔离机制。
- [**微软发布 Autopilot Agent 类别及基于 OpenClaw 的 Microsoft Scout**](https://www.microsoft.com/en-us/microsoft-365/blog/2026/06/02/introducing-microsoft-scout-your-always-on-personal-agent/) — Microsoft 宣布了新的 Agent 类别——Autopilot Agent，并推出其首个产品 Microsoft Scout。它基于 OpenClaw 构建，集成在 Microsoft 365 应用中，能自主协调日程、识别风险，无需每次提示。目前以实验版本通过私人预览和 Frontier 组织提供。
- [**OpenClaw 携手微软将 OpenClaw 引入 Windows 企业环境**](https://aka.ms/OpenClawWindows) — OpenClaw 与 微软 达成合作，企业现可通过 Microsoft Execution Containers 安全运行 OpenClaw，同时 微软 专门为 OpenClaw 打造了一款 Windows Companion App。
- [**微软 Build 2026：Win11 将原生支持 Linux 命令行**](https://blogs.windows.com/windowsdeveloper/2026/06/02/build-2026-furthering-windows-as-the-trusted-platform-for-development/) — 微软在 Build 2026 大会宣布 Win11 开发者优化，系统将正式提供原生 Linux 命令行工具、实验性智能终端。同时，官方发布新一代端侧模型 Aion，并计划推出搭载 英伟达 芯片的专用 AI 开发硬件。
