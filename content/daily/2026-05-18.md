## AI快讯

### 产品更新

- [**知乎数据开放平台提供搜索、热榜和直答服务，支持 Skills 或 API 调用**](https://zhuanlan.zhihu.com/p/2033565515994678331) — 据用户发现，知乎数据开放平台现已提供站内搜索、全网搜索、知乎热榜和知乎直答等能力，并兼容 Skills、API、MCP 等调用方式。目前，注册可获每日 1000 次搜索试用额度。
- [**消息称 Gemini 应用特定模式悄然路由至疑似 3.2 新模型**](https://x.com/chetaslua/status/2055998563721900191) — 据社区用户发现，Google Gemini 应用正逐步更新 UI 并推出"Thinking Level"功能，且在 Fast mode 下运行 Canvas 时，请求疑似已路由至 Gemini 3.2 的新 Flash 模型。上述功能与模型变更目前尚未得到官方确认。
- [**报道称 OpenAI 正在测试 Codex 多设备远程控制与“Locked Use”功能**](https://www.testingcatalog.com/openai-will-let-codex-control-other-desktop-devices-via-computer-use/) — 据报道，OpenAI 正在测试 Codex 的多设备远程控制功能，并计划推出“Locked Use”设置，允许在锁屏状态下执行 Computer Use 任务，但目前上述功能尚未正式开放。

### 行业动态

- [**中国电信发布多档Token套餐 月费9.9元起**](https://mp.weixin.qq.com/s/iwUSgswBqoQ2q29GpjWdaA) — 中国电信推出“试商用”Token套餐，面向个人、家庭及中小微企业开发者提供 9.9元 至 299.9元 不等的月费方案，并计划推出天翼 Token 币及相关权益。
- [**有关部门计划推进全国一体化算力网建设**](https://mp.weixin.qq.com/s/PgLA7DB-hA5MGwos6PITcg) — 媒体报道称，面对激增的AI算力需求，有关部门正计划推进全国一体化算力网建设，将其定位为与电网同等重要的公共基础设施。

### 前沿研究

- [**Google DeepMind 面向亚太区推出 AI for the Planet 加速器计划**](https://blog.google/innovation-and-ai/models-and-research/google-deepmind/accelerator-ai-for-the-planet/) — Google DeepMind 宣布在亚太区推出主题为"AI for the Planet"的首届加速器计划。旨在帮助区域内初创企业、研究团队和非营利组织利用前沿 AI 解决气候、农业及能源等环境问题。
