## AI快讯

### 产品更新

- [**OpenAI 推出 GPT-5.5 Instant，ChatGPT 增强记忆功能**](https://openai.com/index/gpt-5-5-instant/) — OpenAI宣布向所有 ChatGPT 用户推出 GPT-5.5 Instant 作为新默认模型。它在提升智能与图像分析能力的同时大幅减少幻觉，写作风格也更简洁直白。
ChatGPT还增强了记忆功能的效果，并上线了记忆源功能。用户可直接查看和管理个性化回复的具体上下文来源。
- [**Inworld AI 发布 Realtime TTS-2，支持跨语言实时对话**](https://inworld.ai/blog/realtime-tts-2) — Inworld AI发布了新一代实时对话语音模型Realtime TTS-2。
该模型能听取完整对话音频，提供对话感知等核心能力。
它支持在超100种语言中维持同一音色。
- [**Luma 开放 UNI-1.1 系列模型 API，Image Arena 排名全球实验室第三**](https://lumalabs.ai/news/uni-1-1-api) — Luma正式开放UNI-1.1与UNI-1.1-Max的 API。
相关模型采用 decoder-only 自动回归 transformer 架构。
在 Arena.ai 评测中，Luma 凭借该系列模型位列全球实验室第三名。
- [**AI Studio 更新 Build 模块，集成 Nano Banana 与编辑模式**](https://x.com/GoogleAIStudio/status/2051679127824998762) — Google AI Studio 的 Vibe Coding 功能正式上线了编辑模式，并集成了 Nano Banana 图像生成能力。
用户可以直接在界面上进行批注、快速编辑组件。
在生成应用时，系统将自动创建并替换自定义图像资产。
- [**字节跳动 TRAE 上线 SOLO 移动端，支持跨设备协同**](https://mp.weixin.qq.com/s/g0hMDh5PPQWbp_HmXePi0A) — 字节跳动旗下 AI 编程产品"TRAE SOLO"移动端现已正式上线，支持跨设备同步与手机端向电脑派发任务。
同时，官方联合星巴克推出限时咖啡兑换活动。
- [**Augment Code 发布 Augment Cosmos 公开预览**](https://www.augmentcode.com/product/cosmos) — Augment Code 宣布推出 Augment Cosmos 公开预览版。
该产品支持多模型运行，通过引入 Prism 路由技术与专业 Agent，让人类在软件开发生命周期中专注关键节点引导。
现已向 Max 计划用户开放。
- [**谷歌更新 Gemini API 文件搜索，支持多模态检索与精确引用**](https://blog.google/innovation-and-ai/technology/developers-tools/expanded-gemini-api-file-search-multimodal-rag) — Google宣布更新 Gemini API 的 File Search 工具。该工具现支持基于 Gemini Embedding 2 的图文跨模态检索、自定义元数据过滤和精确到页码的引用功能。
- [**Claude 推出系列金融 Agent 模板，多家机构已落地采用**](https://claude.com/solutions/financial-services) — Claude 面向金融行业推出投行推介、月终结账及 KYC 筛查等预构建 Agent 模板。
这些模板内嵌连接器与子 Agent，可作为插件安装到 Claude Cowork 或 Claude Code。
用户也能在 Managed Agents 中直接投产运行。
- [**Google NotebookLM 更新 Mind Maps，移动端开发计划确认**](https://x.com/NotebookLM/status/2051715594697977870) — NotebookLM的Mind Maps功能已推送重大更新。
用户现在能通过输入提示词来定制生成思维导图，还能即时重命名与分享导图，节点导航动画也变得更加丝滑。
- [**Google Finance 推出 AI 关键瞬间功能解析股价波动**](http://finance.google.com/beta) — Google Finance的beta版上线了AI驱动的“关键瞬间”功能。
用户查看一个月及以上的股票走势图时，AI会自动标记并解释股价波动原因。
- [**Google 推出 Pomelli Catalog，免费向全球用户开放体验**](https://x.com/joshwoodward/status/2051698440690889108) — Google 为 Pomelli 项目推出了免费的 Pomelli Catalog 新功能。
全球用户只需在官网添加并编辑产品或服务信息，该 Agent 就会基于品牌 DNA 自动生成产品集合。
系统将持续产出个性化的营销活动与摄影棚级高质量图片。
- [**OpenAI 升级 ChatGPT 广告，开放自助管理及 CPC 竞价**](https://openai.com/index/new-ways-to-buy-chatgpt-ads/) — OpenAI宣布扩展 ChatGPT 广告功能，面向美国广告主推出测试版自助 Ads Manager，并新增单次点击付费的 CPC 竞价模式。
平台还上线了 Conversions API 等工具，在保障对话隐私的前提下提供聚合转化洞察。

### 前沿研究

- [**Subquadratic 推出 SubQ，上下文窗口达 1200 万 token**](https://subq.ai/how-ssa-makes-long-context-practical) — Subquadratic公司宣布推出基于Subquadratic Sparse Attention架构的大语言模型 SubQ。
该模型具备 1200万 token 上下文窗口，在百万 token 任务中实现最高 52 倍预填充加速。
现已开放早期访问，提供 API 与编码 Agent 两种使用方式。
- [**OpenAI 重构 WebRTC，实现语音 AI 低延迟传输**](https://openai.com/index/delivering-low-latency-voice-ai-at-scale/) — 为了让语音AI跟上自然语速，OpenAI发文介绍其重建了 WebRTC 堆栈。
引入薄中继与有状态收发器架构，为 ChatGPT 语音和 Realtime API 实现了大规模低延迟的实时媒体传输。

### 行业动态

- [**OpenRouter 分析 GPT-5.5 涨价，用户成本增 49% 至 92%。**](https://openrouter.ai/announcements/gpt55-cost-analysis) — OpenRouter 发布成本分析指出，由于 token 价格翻倍，用户从 GPT-5.4 切换到 GPT-5.5 后，实际支出成本大幅增加了 49% 到 92%。
- [**OpenAI 与普华永道合作，以 AI Agent 重构财务职能**](https://openai.com/index/openai-pwc-finance-collaboration/) — OpenAI与普华永道合作，帮企业CFO利用AI Agent自动化财务工作流并强化风控。
双方已在OpenAI内部验证，通过Codex把合同处理量提升五倍。
正将这些经验转化为企业部署路径，让财务团队能在既有工具中规模化应用AI Agent。
- [**传 OpenAI 首款 AI 手机 2027 上半年量产 联发科独供**](https://x.com/mingchikuo/status/2051523855286776034) — 分析师郭明錤透露，OpenAI计划最早于2027年上半年量产首款 AI Agent 手机。
该设备预计独家采用联发科定制版 天玑9600 处理器。

### 开源项目

- [**Google DeepMind 为 Gemma 4 系列模型开源了 MTP drafter**](https://blog.google/innovation-and-ai/technology/developers-tools/multi-token-prediction-gemma-4/) — Google DeepMind 为 Gemma 4 系列模型开源了 MTP drafter。
它通过推测解码和共享 KV cache，能够并行预测多个 token。
这一技术在不降低输出质量的前提下，将推理速度最高提升至原来的 3 倍。
- [**OpenClaw 发布 2026.5.4 版，优化启动插件，传将发长期支持版**](https://openclaw.ai/blog/openclaw-rough-week) — OpenClaw 发布了 2026.5.4 版本，重点优化了插件安装体验与 Gateway 启动速度。此次更新同时修复了 Windows 和 Discord 的稳定性问题。
此外，官方计划在五月下旬推出 StableClaw 长期支持版本。
