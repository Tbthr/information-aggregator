## AI快讯

### 产品更新

- [**Claude Code 推出响应速度与可靠性优化更新**](https://x.com/ClaudeDevs/status/2059701687665950859) — Claude Code宣布推出一系列优化更新。新版引入全屏渲染器解决闪屏，实现思考与工具调用的流式输出，并修复上下文压缩报错与MCP连接失败问题。会话遇超大媒体文件现支持自动恢复。
- [**OpenAI 推出 Secure MCP Tunnel 等多项企业级安全与管理功能**](https://developers.openai.com/api/docs/guides/secure-mcp-tunnels) — OpenAI 官方宣布推出 Secure MCP Tunnel、Workload Identity Federation 并扩展 Admin API，旨在帮助企业更安全地集成内部系统并实现平台自动化管理。
- [**腾讯发布 AI 游戏创作平台“代号 Craft”并开启首测**](https://mp.weixin.qq.com/s/csZYzDmJVO__9UsuU_v3nA) — 腾讯发布全新 AI 游戏创作平台“代号 Craft”，支持用户通过自然语言生成 2D 及 3D 游戏原型，目前平台已正式开启首测报名。
- [**Runway 推出 Runway MCP 服务**](https://runwayml.com/news/mcp) — Runway 宣布推出 Runway MCP。通过该服务，用户可在 Claude、Cursor 等任何兼容 MCP 的 Agent 内直接生成图像与视频。
- [**NotebookLM 推出 Google Drive 文件自动同步功能**](https://x.com/joshwoodward/status/2059665697618493706) — NotebookLM 宣布推出 Google Drive 文件自动同步功能。该功能目前已启动推送，首批面向 10% 的用户开放，并将在后续逐步扩大范围。
- [**Sesame 推出 iOS 应用预览版，首发多款个人 Agent**](https://apps.apple.com/us/app/sesame-personal-agents/id6756329076%E2%80%A6) — Sesame 推出 iOS 应用预览版，首发多款用于日常对话的个人 Agent。应用正逐步向多国开放，Android 版已在路线图中。
- [**YouTube 引入自动检测机制主动标记 AI 生成视频**](https://blog.youtube/news-and-events/improving-ai-labels-viewers-creators/) — YouTube 宣布引入自动 AI 检测系统，将主动为“逼真的 AI”生成内容添加标签，该标签会直接显示在长视频与 Shorts 播放界面上，同时平台将在未来几周向所有 18 岁以上频道全面开放 AI 深度伪造肖像检测技术。
- [**千问升级“拍照问健康”功能**](https://mp.weixin.qq.com/s/QVo8TJiwymYNkqZlbOzo4A) — 千问APP升级拍照问健康功能，支持上传检查单等图像，系统会圈定异常并结合知识库推理给出参考。官方邀请三甲医生验证了其具备鉴别诊断等专业能力。
- [**Claude 推出 AI Agent 零信任架构指南**](https://claude.com/blog/zero-trust-for-ai-agents) — Claude 官方发布了面向企业自主 **AI Agent** 的零信任安全框架指南，提出采用三层安全架构与八阶段实施工作流，以应对相关新型威胁。
- [**Meta推出三大应用Plus订阅服务并测试Meta One计划**](https://www.aibase.com/zh/news/28396) — Meta近日宣布面向全球推出Instagram、Facebook和WhatsApp的Plus订阅服务，并同步在多国开启整合AI与企业服务的“Meta One”测试。

### 前沿研究

- [**OpenAI 发文介绍与 Thrive Holdings 联合开发基于 Codex 的自我改进税务 Agent**](https://openai.com/index/building-self-improving-tax-agents-with-codex/) — OpenAI 发文介绍与 Thrive Holdings 联合开发了税务 Agent“Tax AI”。该系统基于 Codex 构建，通过将审查人员的纠错转化为结构化信号，实现在真实税务处理中的自主迭代与自我改进。
- [**Anthropic 报告：仅 20% 社会科学家常态化使用编码 Agent**](https://www.anthropic.com/research/coding-agents-social-sciences) — Anthropic 发布针对 1260 名定量社会科学家的调查报告。数据显示，81% 的受访者曾尝试 AI 聊天机器人，但仅 20% 每周常态化使用 Claude Code 等 编码 Agent。
