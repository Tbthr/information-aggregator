## AI快讯

### 产品更新

- [**谷歌 Gemini 上线新功能，正式支持对话直接生成多种格式文件**](https://blog.google/innovation-and-ai/products/gemini-app/generate-files-in-gemini/) — Gemini宣布正式上线了全新的文件生成功能。用户现在只需在聊天界面输入一句话，就能直接生成并下载 PDF、Word 或 Excel 等多种格式的完整文件。全程无需手动排版或复制粘贴。
- [**Google AI Studio build 模式上线搜索与多会话功能**](https://x.com/GoogleAIStudio/status/2049519191389073633) — Google AI Studio 为 build 模式推出了两项新功能。现在可以通过新增的 web search 实时联网获取最新文档，还能利用 multi-chat 在不同想法的独立会话间自由切换。
- [**Cursor 发布 TypeScript SDK 公测**](https://x.com/cursor_ai/status/2049499866217185492) — Cursor 正式推出了官方 TypeScript SDK 公测版。开发者现在可以通过它将 Cursor 强大的 Agent 运行时直接集成到自己的应用中，支持在本地或云端专属沙箱运行任务。
- [**OpenAI Codex 接入 Figma 与 Supabase，拟增强长任务能力**](https://x.com/OpenAIDevs/status/2049605820351230158) — OpenAI的Codex迎来多项重要更新，现已正式接入Figma的FigJam与Supabase，能将实施计划转为可视化看板并跨数据库执行任务。同时，OpenAI准备发布新变更来优化Codex的长周期运行能力。
- [**Zed 团队发布 Zed 1.0 版**](https://zed.dev/blog/zed-1-0) — Zed团队正式发布了代码编辑器Zed的1.0版本。这款由Atom创始人打造的编辑器现已支持三大操作系统。它集成了Git和SSH等完整开发功能，原生支持多种AI助手协同工作。
- [**ima 正式上线知识 Agent 模式，支持全平台及自定义模型接入**](https://mp.weixin.qq.com/s/fxaw_xjIn5iyBXKSQtswDg) — ima.copilot正式上线了知识 Agent 模式 copilot，目前用户可在全平台申请创建自己的专属 Agent。同时，ima正式允许用户自行接入各类模型的 API KEY。
- [**AWS 正式发布 Amazon Quick，定位为跨应用 AI 办公助理**](https://www.aboutamazon.com/news/aws/amazon-quick-desktop-ai-assistant) — AWS 发布了 Amazon Quick 桌面应用，这是一款跨应用的 AI 办公助理。它能连接本地文件、日历邮件及各类第三方工具。通过长期记忆和主动提醒功能帮用户高效办公。
- [**ElevenLabs 发布 ElevenMusic 平台，支持创作者变现**](https://x.com/ElevenLabs/status/2049493164902187301) — ElevenLabs宣布发布ElevenMusic平台，用户可以通过这个平台使用 AI 创作或者混音音乐作品。用户还能通过发布作品来获得收入。
- [**xAI 于 Grok Web 推出 Imagine Agent 图像处理功能**](https://x.com/techdevnotes/status/2049605616650408019) — 据用户发现，xAI 在 Grok 网页端推出了 Imagine Agent Mode，提供快速动画、群组图片和裁剪等多项图像处理功能。
- [**Hermes Agent 集成 ComfyUI 控制能力，支持工作流管理**](https://x.com/NousResearch/status/2049584595465572752) — Nous Research 宣布 Hermes Agent 现已原生集成对 ComfyUI 的控制能力。用户只需运行更新命令，就能直接在本地或云端轻松管理和运行复杂的 ComfyUI 工作流。

### 前沿研究

- [**Anthropic 发布内省适配器，支持模型审计隐藏不对齐行为**](https://alignment.anthropic.com/2026/introspection-adapters/) — Anthropic 发布最新研究，推出了一款名为 内省适配器 的工具。它能引导语言模型主动报告在训练中学会的隐藏不良行为。并在针对安全后门等隐蔽威胁的检测测试中表现优异。为 人工智能 的安全对齐审计提供了全新手段。

### 行业动态

- [**Exa Labs 合作 Google，搜索集成至 Vertex 及 Gemini 企业版**](https://exa.ai/blog/exa-google-cloud) — Exa Labs与谷歌达成合作，将其AI搜索引擎接入谷歌云生态。Exa现已通过Vertex AI的Grounding功能为Gemini模型提供实时搜索结果。

### 开源项目

- [**腾讯混元开源 Hy-MT1.5-1.8B 端侧翻译模型**](https://modelscope.cn/collections/AngelSlim/Hy-MT15-18b-quant) — 腾讯混元团队开源了专为手机端打造的翻译模型Hy-MT1.5-1.8B。只需不到 600兆 的存储空间即可在离线环境下流畅运行，支持 三十多种 语言。官方同步推出了安卓端完全离线演示应用。
- [**IBM 发布 Granite 4.1 系列开源模型**](https://research.ibm.com/blog/granite-4-1-ai-foundation-models) — IBM 发布了采用 Apache 2.0 开源许可的 Granite 4.1 系列模型。全系列涵盖了多种参数规模的语言模型。此外，该系列还包括视觉、语音、安全检测和文本嵌入等专用模型。
- [**蚂蚁百灵开源 Ling-2.6-1T 模型**](https://huggingface.co/inclusionAI/Ling-2.6-1T) — 蚂蚁百灵团队宣布开源此前发布的模型 Ling-2.6-1T。目前该模型已在 Hugging Face 和 ModelScope 平台正式上线开源。此外，在 OpenRouter 上还将提供延长 一周 的免费调用服务。
- [**Qwen 团队开源 FlashQLA 优化 GDN 架构获 2 至 3 倍加速**](https://qwen.ai/blog?id=flashqla) — Qwen团队正式开源了高性能线性注意力算子库 FlashQLA，该算子库在 NVIDIA Hopper 架构上实现了 2至3倍 的前向加速与 2倍 的反向加速。这一性能突破大幅提升了大模型预训练与端侧 Agent 推理的运行效率，为相关领域的计算优化提供了新的强力支持。

- [**科大讯飞发布星火 X2-Flash，基于华为昇腾 910B 训练**](https://mp.weixin.qq.com/s?__biz=MzU5NzczMDI5OA==&amp;mid=2247520337&amp;idx=1&amp;sn=47be8f669f0deb9ed2e4a03eb65d1c1e&amp;chksm=ffc78660f842b5c5f2f7dfd0fc56d2c4adade662e984dec225e7a55c748492ea9f8fe2e44ac0#rd) — 科大讯飞正式发布了星火X2-Flash模型，并同步开放 API 接口。这款模型总参数量为 30B，完全基于华为昇腾910B集群训练，重点提升了智能体与代码能力。
- [**百度发布文心一言 5.1 预览版**](https://yiyan.baidu.com/blog/zh/posts/ernie-5.1-preview-0430-release-on-lmarena/) — 百度宣布文心大模型ERNIE-5.1-Preview现已上线。它在大模型竞技场LMArena文本榜拿下国内第一、全球第十三的成绩。目前可在文心一言官网体验。
