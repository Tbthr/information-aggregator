## AI快讯

### 开源项目

- [**SpaceXAI 完成 Grok V9-Medium 模型训练并预告开源计划**](https://x.com/elonmusk/status/2058787384364265734) — Elon Musk 宣布，1.5T 参数的 Grok 基础模型 V9-Medium 已完成训练并进入微调阶段，几天后启动强化学习，预计在 2 到 3 周内公开发布，同时他还回应网友评论称，将在 今年年底开源 0.5T 参数的模型。
- [**OpenBMB 发布 MiniCPM5-1B 开源模型**](https://huggingface.co/openbmb/MiniCPM5-1B) — OpenBMB 开源了 MiniCPM5-1B 模型。该模型具备混合推理能力，登顶了 Artificial Analysis 小模型榜单，成为 2B 以下最强模型。

### 产品更新

- [**Grok Build 面向 SuperGrok 和 X Premium+ 用户开放**](https://x.com/xai/status/2058973760708091907) — SpaceXAI 宣布 Grok Build 现已开启 Beta 测试，并面向所有 SuperGrok 和 X Premium+ 用户开放，支持 Plan Mode、Imagine 媒体生成及通过 CLI 构建自动化任务。
- [**Google Antigravity 新增 Gemini 3.5 Flash (Low)**](https://x.com/antigravity/status/2058741814237241812) — Antigravity 宣布引入 Gemini 3.5 Flash (Low) 选项，官方称其生成 token 数较 Medium 版减少约 45% 以优化简单任务，同时已重置所有付费用户的配额。
- [**ima 宣布全面开放 copilot 并上线 Skill 发布功能**](https://mp.weixin.qq.com/s/R22ySmNraCJ4W4o2nbJu1w) — ima 宣布取消排队限制，全面开放具备记忆与知识库的 Agent “copilot”，同时知识号同步上线发布 Skill 功能，用户需将应用更新至最新版即可体验。

### 行业动态

- [**华为发布韬(τ)定律：麒麟芯片将首发逻辑折叠技术**](https://mp.weixin.qq.com/s/txF-C8pXnmGwqNLFEDep8A) — 华为近日提出半导体发展新原则——韬定律。该定律主张以“时间缩微”替代传统的“几何缩微”，通过“逻辑折叠”等创新技术压缩信号传播时延，从而持续提升晶体管密度与系统性能。

### 社媒精选

- [**社区反馈 Claude 注册疑似取消手机验证并重开申诉**](https://linux.do/t/topic/2245547) — 据社区用户反馈，近期 Claude 账号注册疑似取消了手机号验证，同时还开放了此前被封禁账号的申诉入口。
