## AI快讯

### 产品更新

- [**Codex 为 Windows 端上线 Computer use 与远程控制功能**](https://x.com/OpenAI/status/2060428604727771421) — OpenAI 发布了 Codex 重大更新，Computer use 功能登陆 Windows 平台。可前台操控桌面应用进行测试和调试，并支持通过移动端或 Mac 远程控制 Windows 设备。此外，本次更新还为后台 Agent 引入了便于识别的像素头像，新增了对所有历史聊天内容的搜索支持，并上线了包含 Token 活动详情的新个人资料面板。同时，iOS 版 ChatGPT 也迎来了 Codex 专项更新，引入了 /side 临时对话等高级功能。
- [**ChatGPT 更新 GPT-5.5 Instant 模型并推出长对话目录功能**](https://help.openai.com/en/articles/6825453-chatgpt-release-notes?utm_source=chatgpt.com) — OpenAI 更新了 GPT-5.5 Instant 模型，让回复更加自然，并移除了 Canvas 功能，改为直接在回复中提供 writing blocks 与 code blocks。同时，ChatGPT 为包含五条以上回复的长对话新增了自动目录功能。此外，官方宣布 GPT-4.5 与 o3 将分别于 2026年6月 和 8月 从 ChatGPT 下线，API 调用不受影响。
- [**谷歌Gemini调整配额规则：Ultra会员视频生成量翻倍**](https://x.com/joshwoodward/status/2060171610922058142) — 谷歌针对Gemini App配额消耗过快宣布多项修复。官方已修复Omni视频Bug，Ultra会员视频生成配额立即翻倍。Flash-Lite模型的调用将不再扣除配额。同时官方正为Deep Research设计详细用量明细。

### 行业动态

- [**OpenAI推出Rosalind Biodefense项目，加速生物防御与防疫**](https://openai.com/index/strengthening-societal-resilience-with-rosalind-biodefense/) — OpenAI宣布推出Rosalind Biodefense项目，向全球合格开发者赞助生命科学模型GPT-Rosalind，并向特定政府扩大受信任访问，以加速构建生物防御与大流行病防范能力。
- [**报道称智谱AI正开发终端硬件产品，首款设备已获工信部试用证**](https://mp.weixin.qq.com/s/m06yIsnYFaGyx6MNlrnk2Q) — 据报道，智谱AI正研发多款终端硬件产品，其中型号 ZAI-P1 已获 工信部 进网试用证，由早教代工厂生产，推测其或将进军教育终端。

### 开源项目

- [**Hermes Agent 推出 Tool Search 并发布 v0.15.1 修复补丁**](https://hermes-agent.nousresearch.com/docs/user-guide/features/tool-search) — Hermes Agent 推出 Tool Search 功能，按需加载 MCP 与非核心插件工具的 schema 以减少上下文，同时官方发布 v0.15.1 补丁，修复了 Dashboard 无限重载等 Bug。
- [**阿里云百炼宣布推出并开源百炼 CLI**](https://github.com/modelstudioai/cli) — 阿里云百炼宣布开源百炼 CLI。开发者仅需一行命令，即可让 AI Agent直接调用百炼提供的多模态生成、知识库及记忆等全栈 AI 能力。
