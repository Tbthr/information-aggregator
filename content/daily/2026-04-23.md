## AI快讯

### 开源项目

- [**Qwen 团队开源 Qwen3.6-27B 模型**](https://qwenlm.github.io/zh/blog/qwen3.6-27b/) — Qwen团队正式发布了拥有270亿参数的开源稠密模型 Qwen3.6-27B。
该模型原生支持多模态及思考模式，编程能力全面超越了前代开源旗舰 Qwen3.5-397B-A17B。
目前开源权重已在各大平台开放，用户可通过 Qwen Studio 直接体验。
- [**OpenAI 发布开源隐私过滤模型，支持本地检测脱敏**](https://openai.com/index/introducing-openai-privacy-filter/) — OpenAI 发布了名为 OpenAI Privacy Filter 的开放权重模型。
该模型总参数量为 1.5B，激活参数为 50M。
它主要用于检测和脱敏文本中的个人身份信息。
- [**Google 开源 DESIGN.md 规范**](https://github.com/google-labs-code/design.md) — Google 开源了其专门用于帮助 AI 编程智能体准确理解和执行项目设计的 DESIGN.md 规范。
该规范结合了机器可读的设计 token 与人类可读的文本说明，并同步推出了配套的命令行工具。
- [**Hugging Face 发布 ml-intern 开源 AI Agent 自动化大模型后训练**](https://github.com/huggingface/ml-intern) — Hugging Face 发布了名为 ml-intern 的开源 AI Agent，官方称这款工具能够代替人类研究员。
该工具旨在全自动完成大模型的后训练任务。

### 产品更新

- [**Claude 开放 ultrareview 功能，订阅用户享三次免费审查**](https://x.com/ClaudeDevs/status/2046999435239133246) — Claude 官方宣布，Claude Code 的代码审查功能 /ultrareview 现面向 Pro 和 Max 用户提供截至 5月5日 的 3次 免费使用机会。
- [**Zed 推出 Parallel Agents，支持多 Agent 并行运行**](https://zed.dev/blog/parallel-agents) — Zed编辑器在最新版本中推出了Parallel Agents功能，允许开发者在单一窗口内并行编排多个Agent。
用户可以通过全新的侧边栏对它们进行集中管理。
- [**火山方舟上线 GLM-5.1 模型，ArkClaw 集成 Hermes Agent**](https://mp.weixin.qq.com/s/PGH22zYObkTHVZRXr9-I4Q) — 火山方舟 Coding Plan 上线了 GLM-5.1 和 Kimi K2.6 等模型，同时 ArkClaw 平台推出了 Hermes Agent 插件。
- [**OpenAI 宣布 ChatGPT 上线 Workspace agents 功能**](https://openai.com/business/workspace-agents/) — OpenAI面向企业版和教育版等订阅用户在 ChatGPT 中推出了 Workspace agents 功能。
通过该功能，团队可一次性构建具备特定工作能力的 Agent 并在工作区内共享。
且能将其设定为按计划自动执行重复性任务。
- [**OpenAI 发布 ChatGPT for Google Sheets 插件**](https://chatgpt.com/apps/spreadsheets/) — OpenAI 推出了 ChatGPT for Google Sheets 插件。
用户现在可以直接在 Google Sheets 里向 ChatGPT 提问或完成各类操作。
- [**Google 推出 Gemini Enterprise Agent Platform 统一 AI 技术栈**](https://blog.google/innovation-and-ai/infrastructure-and-cloud/google-cloud/gemini-enterprise-agent-platform) — Google 官方宣布推出全新的 Gemini Enterprise Agent Platform。
这个一站式平台能帮助技术团队大规模构建和管理 AI 智能体，并让普通业务人员通过应用实现零代码使用与创建智能体。
- [**谷歌宣布 Workspace Intelligence 全面上线**](https://x.com/demishassabis/status/2046994882150154369) — 谷歌宣布内置于 Workspace 办公套件的 Workspace Intelligence 全面上线。
Workspace Intelligence 整合了邮件和云盘等应用数据，为 Docs 和 Sheets 带来了由 Gemini 驱动的智能写作、风格模仿以及自动生成表格等新功能。
这一举措标志着 谷歌 正在全面发力企业办公自动化领域。
- [**Google 升级 Google Earth AI 及 Google Maps Platform**](https://mapsplatform.google.com/resources/blog/three-new-ways-to-build-with-real-world-imagery-and-ai/) — 谷歌宣布对 Google Earth AI 和 Google Maps Platform 进行重大扩展。
此次更新增加了多项基于卫星与街景图像的人工智能分析工具，帮助公共部门和企业更好地理解和分析物理世界。
- [**Claude 官方详解 MCP 连接方案，SDK 月下载超三亿次**](https://claude.com/blog/building-agents-that-reach-production-systems-with-mcp) — Claude 官方发布博文指出，随着生产级 Agent 向云端迁移，MCP 已经成为连接外部系统最关键的方式。
官方结合相关数据，详细分享了构建高效云端 MCP 架构的设计模式与最佳实践。

### 前沿研究

- [**谷歌推出 TorchTPU 支持 PyTorch 原生运行 TPU**](https://developers.googleblog.com/torchtpu-running-pytorch-natively-on-tpus-at-google-scale/) — 谷歌宣布推出 TorchTPU 技术栈，让 PyTorch 能够原生且高效地运行在 TPU 硬件上。
- [**Google Cloud 发布第八代 TPU，采用双芯片架构设计**](https://blog.google/innovation-and-ai/infrastructure-and-cloud/google-cloud/eighth-generation-tpu-agentic-era/) — Google Cloud宣布推出第八代定制 AI 芯片 TPU 系列，包含专攻模型训练的 TPU 8t和专攻推理的 TPU 8i。
这两款芯片均首次采用谷歌自研的 Axion ARM架构 CPU，并配备第四代液冷技术。
谷歌计划在今年晚些时候全面提供这两款芯片服务。

### 行业动态

- [**小米发布 MiMo-V2.5 系列四款模型，两款模型将开源**](https://platform.xiaomimimo.com/docs/news/v2.5-news) — 小米宣布开启MiMo-V2.5系列大模型公测，该系列包含MiMo-V2.5、V2.5-Pro、V2.5-TTS Series及V2.5-ASR四款模型。
其中，MiMo-V2.5-Pro专为复杂的长程Agent任务打造。官方数据显示其在通用智能体能力、复杂软件工程等维度已能媲美前沿闭源模型。
而MiMo-V2.5则是为Agent场景而生的原生全模态模型。相关模型已上线API和Token Plan。
其中MiMo-V2.5和V2.5-Pro两款模型即将开源。
- [**GLM 调整老套餐计划，月底强制终止续订并送两月权益**](https://docs.bigmodel.cn/cn/coding-plan/transition) — 智谱宣布，将在4 月 30 日起停止 GLM Coding Plan无周限额老套餐的自动续订，并向受影响用户自动赠送2 个月同等级新套餐作为补偿。
尽管当前订阅周期的价格和权益不受影响，但针对此次调整，不少社区用户认为此举有背刺早期支持者的嫌疑。
- [**OpenAI 规划 2030 年实现 30GW 算力**](https://x.com/OpenAINewsroom/status/2046951726683455866) — OpenAI官方表示，在目前已确定超过 8GW 算力规模的基础上，计划到 2030 年将计算能力提升至 30GW，以满足快速激增的需求。
- [**马斯克宣布建 Terafab 超级芯片工厂，确认采用英特尔 14A 工艺**](https://www.reuters.com/business/autos-transportation/tesla-ceo-musk-says-company-plans-use-intels-14a-process-terafab-2026-04-22/) — 据报道，特斯拉、SpaceX和xAI将合资在德克萨斯州建设名为 Terafab 的超级芯片工厂。
该项目计划采用英特尔的14A制程工艺。
- [**媒体称腾讯阿里洽谈投资 DeepSeek，估值目标超两百亿美元**](https://www.theinformation.com/articles/tencent-alibaba-talks-invest-deepseek-20-billion-plus-valuation) — 据报道，腾讯和阿里巴巴正洽谈投资DeepSeek。这笔首次外部融资的估值目标已因投资者反响强烈而提升至超过两百亿美元。
在产品端，非官方人员披露了疑似 DeepSeek V4 模型的详细参数。同时，有用户观察到其 API 疑似出现了新模型的上线测试与回滚变动。
- [**Meta 被曝采集员工操作数据训练 AI**](https://www.reuters.com/sustainability/boards-policy-regulation/meta-start-capturing-employee-mouse-movements-keystrokes-ai-training-data-2026-04-21/) — 据 路透社 报道，Meta 正在美国员工的办公电脑上部署内部追踪工具。
该工具负责收集鼠标点击和键盘输入等操作数据。这些数据将被用于训练能够自动执行办公任务的 AI 模型。
