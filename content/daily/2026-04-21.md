## AI快讯

### 开源项目

- [**月之暗面正式发布并开源了 Kimi K2.6 模型**](https://www.kimi.com/blog/kimi-k2-6) — 月之暗面正式发布并开源了 Kimi K2.6 模型。该模型采用混合专家架构，总参数量1T，激活参数32B，上下文长度支持 256K。官方称这款模型在代码编写、长程任务执行和Agent集群能力方面均达到行业领先水平，并在多项基准测试中表现优于多款主流闭源模型。目前该模型已通过官方渠道以及各大第三方平台全面上线。
- [**阿里发布 Fun-ASR1.5，支持 30 种语言及方言识别**](https://mp.weixin.qq.com/s/xPEstUiJ4IC3BOFVsdvRLA) — 阿里发布了端到端语音识别大模型 Fun-ASR1.5。该模型无需预设语种就能自动切换并精准识别 30 种语言。不仅攻克了多种中文方言和古诗词的识别难题，还能智能添加标点并将口语自动转换为规范文本。目前已上线 阿里云百炼并在 魔搭提供体验。

### 产品更新

- [**Google AI Studio 正式接入 Google AI 订阅**](https://blog.google/innovation-and-ai/technology/developers-tools/google-one-ai-studio/) — Google宣布，AI Pro 和 AI Ultra订阅用户现已可以在Google AI Studio里使用订阅权益。无需绑定API密钥，就能访问 Nano Banana Pro在内的模型并获得更高的调用额度。
- [**OpenAI 为 Codex 上线 Chronicle，支持屏幕上下文记忆功能**](https://developers.openai.com/codex/memories/chronicle) — OpenAI为 Codex推出了名为 Chronicle的实验性新功能，它能通过捕获用户的屏幕画面来自动生成记忆，省去重复沟通背景信息的麻烦。不过官方也警告，这项功能不仅会快速消耗额度，还存在提示注入攻击以及本地明文存储带来的隐私风险。目前该功能仅面向部分地区的 macOS 版 ChatGPT Pro 订阅用户开放。
- [**Qwen 团队发布 Qwen3.6-Max-Preview 模型**](https://mp.weixin.qq.com/s/DKxrFnBwisNjjOnFQRqsqA) — Qwen团队推出了下一代旗舰模型的早期预览版Qwen3.6-Max-Preview。该模型在智能体编程、世界知识等方面显著增强，并在六项主要编程基准上拿下了最高分。目前已经可以在Qwen Studio上直接体验该模型。后续也将提供 API 调用。
- [**ZCode 发布 1.0.0，推出自研ZCode Agent**](https://zcode-ai.com/cn/changelog/1.0.0) — ZCode 发布了 1.0.0 版本，采用全新架构大幅提升了桌面端的流畅度与稳定性。此次更新还支持一键导入 Claude Code 数据，推出了集成多种命令行工具的自研 ZCode Agent。新版本并全面支持远程 SSH 和 Docker 环境开发。
- [**阿里云百炼推出 Token Plan 团队版服务**](https://help.aliyun.com/zh/model-studio/token-plan-overview) — 阿里云百炼推出了名为Token Plan 团队版的AI 大模型订阅服务。该服务通过Credits统一计费，整合了多款主流模型。目前提供198 元至1398 元三档包月套餐。
- [**谷歌将 Chrome AI 功能扩展至亚太七国新市场**](https://blog.google/products-and-platforms/products/chrome/chrome-expands-apac/) — Google宣布将 Chrome 浏览器的最新 AI 功能扩展至澳大利亚和日本等七个亚太地区新市场，桌面端和iOS用户现可通过内置的 Gemini 助手总结网页内容或直接调用日历等其他应用。
- [**Claude Cowork 推出 Live Artifacts 功能**](https://x.com/felixrieseberg/status/2046334841663389792) — Anthropic旗下的 Claude推出了 Cowork功能的Live Artifacts更新。这项更新能让你创建的仪表盘和报告，在每次打开时自动从Connectors获取最新数据。

### 行业动态

- [**GitHub 调整 Copilot 订阅计划**](https://github.blog/changelog/2026-04-20-changes-to-github-copilot-plans-for-individuals/) — GitHub宣布调整 Copilot 个人版计划，目前已暂停部分付费版本的新用户注册并收紧了使用限额。受影响的现有付费用户可以在公告发布后一个月内联系支持团队申请免责退回4 月费用。
- [**Anthropic 与 Amazon 深化合作，十年投入超千亿美元算力**](https://www.anthropic.com/news/anthropic-amazon-compute) — Anthropic与Amazon宣布深化战略合作，Amazon向Anthropic追加50亿美元投资。而Anthropic则承诺未来十年向AWS基础设施投入超1000亿美元。此次合作将使用Amazon自研的Trainium芯片，为Claude锁定高达5GW的新增算力。
- [**马斯克宣布 Grok 将推出办公软件插件**](https://x.com/elonmusk/status/2046016151352930396) — Elon Musk 宣布，xAI 旗下人工智能产品 Grok 即将推出适用于 Word 和 PowerPoint 等办公软件的插件。
- [**据称 OpenAI 将发新图像模型 助 ChatGPT 破十亿用户瓶颈**](https://www.theinformation.com/newsletters/ai-agenda/openai-takes-aim-google-new-image-model) — 据媒体报道，OpenAI 即将发布一款全新的图像模型 GPT-Image-2。该模型在视觉逼真度和文本渲染能力上均有显著提升。OpenAI 期望借此推动 ChatGPT 打破目前 10亿 周活跃用户的增长瓶颈。
- [**据称 NSA 使用 Anthropic Mythos 模型扫描漏洞**](https://techcrunch.com/2026/04/20/nsa-spies-are-reportedly-using-anthropics-mythos-despite-pentagon-feud) — 据报道，Anthropic 与特朗普政府的关系正在回暖。美国国家安全局正在使用 Anthropic 未公开发布的 Mythos 模型来扫描网络漏洞。
- [**媒体称 Google DeepMind 组建专班强化 Gemini 编程**](https://the-decoder.com/google-builds-elite-team-to-close-the-coding-gap-with-anthropic/) — 据报道，为了缩小在 AI 编程领域与 Anthropic 的差距，Google DeepMind 组建了一支由联合创始人 谢尔盖·布林 督战的专门团队。该团队旨在全面强化 Gemini 模型的复杂编程能力与智能体水平。
