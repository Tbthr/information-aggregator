## AI快讯

### 产品更新

- [**千问发布新一代旗舰 Qwen3.7-Max，即将上线百炼 API**](https://qwenlm.github.io/zh/blog/qwen3.7/) — Qwen 团队正式发布面向 Agent 时代的旗舰模型 Qwen3.7-Max，主打编程、办公自动化与长周期自主执行能力。官方称，该模型在编程 Agent、通用 Agent 及高难度推理等多个评测维度表现领先，即将通过 阿里云 百炼 API 上线。
- [**千问发布实时同传模型 Qwen3.5-LiveTranslate**](https://qwen.ai/blog?id=qwen3.5-livetranslate) — 千问发布实时同传模型 Qwen3.5-LiveTranslate-Flash。该模型支持60种语言音频输入及29种语言语音输出，并具备实时跨语言音色克隆能力，现已开放 Demo 体验。
- [**Stability AI 发布 Stable Audio 3.0 系列音频模型**](https://stability.ai/stable-audio) — Stability AI 发布 Stable Audio 3.0 系列音频生成模型，提供多个规格，其中小号与中号版本以开放权重形式发布，支持最长 6 分钟的音频生成，可在一定限制下免费商用。
- [**Codex 重置使用限制**](https://x.com/thsottiaux/status/2056851041132696025) — 疑似迎接 Google I/O 召开，OpenAI 于5月20日早上宣布重置了 Codex 的速率限制。
- [**Hermes agent 上线 SpaceXAI 网络搜索功能**](https://hermes-agent.nousresearch.com/docs/user-guide/features/web-search#xai-grok) — NousResearch 的 Hermes agent 现已支持调用 Grok 执行网络搜索，相关功能文档已同步上线。
- [**OpenClaw 发布 2026.5.19 更新**](https://github.com/openclaw/openclaw/releases/tag/v2026.5.19) — OpenClaw 发布 2026.5.19 更新，Android Talk Mode 迎来实时化升级，并优化了 Mac 设置界面与无头设备的 SpaceXAI 登录体验。
- [**Hugging Face 上线 Hardware 页面及模型参数筛选功能**](https://huggingface.co/datasets?benchmark=benchmark:official&amp;sort=trending) — Hugging Face 宣布为 Dataset Leaderboard 新增按模型参数量级筛选基准测试结果的功能，并推出用于展示开源 AI 社区真实硬件使用情况的 Hugging Face Hardware。
- [**Google 发布跨平台 Agent Ask Advisor 及多项广告 AI 更新**](https://blog.google/products/ads-commerce/ask-advisor/) — Google 连续发布多项广告与营销 AI 更新，宣布跨产品 Agent Ask Advisor 现已面向英文账户推出测试版，Asset Studio 即将集成 Gemini Omni 模型，并在 AI Mode 中测试多种由 Gemini 驱动的新型广告格式。

### 前沿研究

- [**OpenAI宣布内部通用模型自主攻克Erdős问题，证明已获外部验证**](https://openai.com/index/model-disproves-discrete-geometry-conjecture/) — OpenAI宣布其通用推理模型自主解决平面单位距离问题，成功推翻近80年的离散几何核心猜想，标志着AI首次独立解决数学界著名开放问题。
- [**Z.ai 等联合部署 ZCube 网络架构：推理吞吐提升 15%**](https://x.com/Zai_org/status/2057216685040443743) — 智谱与相关机构联合开发并在生产环境部署了 ZCube 网络架构。官方数据显示，该架构使 GLM-5.1 推理集群的网络硬件成本降低 33%，平均吞吐量提升 15%，目前已稳定运行超 两周。

### 开源项目

- [**Cohere 发布开源模型 Command A+**](https://cohere.com/blog/command-a-plus) — Cohere 发布开源大模型 Command A+。该模型采用 218B 总参数和 25B 激活参数的 MoE 架构，它整合了前代视觉与推理能力，专精 Agentic 任务。
- [**NVIDIA 推出 SANA-WM 分钟级视频生成模型**](https://nvlabs.github.io/Sana/WM/) — NVIDIA 开源世界模型 SANA-WM，支持基于单张图像与摄像机轨迹，在单块 GPU 上生成 720p 分辨率的 60 秒可控视频。
