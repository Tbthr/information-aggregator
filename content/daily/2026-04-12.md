## AI快讯

### 产品更新

- [**OpenAI 澄清 ChatGPT Pro 额度规则，承认定价页面表述有误将更新**](https://x.com/thsottiaux/status/2043075353242218768) — OpenAI 工作人员承认 ChatGPT 定价页面表述有误，明确了 ChatGPT Pro 订阅计划的额度：在持续至 5月31日 的促销期内，100美元 和 200美元 的 Pro 订阅实际分别拥有 10倍 和 20倍 于 Plus 订阅的额度。
- [**阿里云百炼停售并清退 Coding Plan Lite 套餐**](https://www.aliyun.com/notice/118175?spm=a2c4g.11186623.0.0.31d84c4dSe5LUM) — 阿里云发布公告宣布，因产品策略调整，4月13日起将正式停止大模型服务平台百炼的Coding Plan Lite基础套餐的续费与升级，已购用户的现有权益不受影响，可正常使用至服务到期。
- [**GitHub Copilot 宣布将引入全新速率限制**](https://github.blog/changelog/2026-04-10-enforcing-new-limits-and-retiring-opus-4-6-fast-from-copilot-pro/) — GitHub 宣布将在未来几周内为 Copilot 引入全新速率限制以应对高并发压力。同时，官方即日起对 Pro+ 用户停用 Opus 4.6 Fast 模型。
- [**OpenClaw 发布 2026.4.10 版，优化记忆功能**](https://x.com/openclaw/status/2042811598058742012) — OpenClaw 正式发布了 2026.4.10 版本。此次更新重点推出了提升交互记忆的 Active Memory 以及面向开发者的 Codex app-server harness，同时在底层安全防护方面进行了升级。

### 开源项目

- [**MiniMax 今日正式开源 MiniMax-M2.7 模型**](https://huggingface.co/MiniMaxAI/MiniMax-M2.5/discussions/53#69dacdbf7cf2e018a77e48b9) — 据 MiniMax 工作人员消息，MiniMax 已定于今天 9:00 正式开源 MiniMax-M2.7 模型。
截至本期内容发布前，模型仓库尚未正式开放。

### 行业动态

- [**OpenAI 响应 Axios 事件，撤销证书敦促四款应用升级**](https://openai.com/index/axios-developer-tool-compromise/) — OpenAI 因第三方开发工具 Axios 遭遇供应链安全事件，正在全面撤销并轮换其安全证书。
为防范风险，OpenAI 强烈建议 Mac 用户立刻将 ChatGPT、Codex、Atlas 以及 Codex CLI 这四款应用升级至最新版本。
- [**OpenAI CEO 住所遭袭，嫌疑人被捕，Altman 发文回应**](https://blog.samaltman.com/2279512) — OpenAI CEO Sam Altman 位于旧金山的住宅遭到一名 20岁 男子投掷燃烧瓶袭击，导致大门起火，所幸未造成伤亡。该嫌疑人在作案后前往 OpenAI 总部威胁纵火时被警方逮捕。
针对此次袭击，Altman 发文指出，煽动性文章加剧了公众对 AI 的焦虑。他借此呼吁全行业缓和言辞，以减少冲突，并重申 AI 必须实现民主化，其控制权与塑造未来的权力绝不能过度集中于少数实验室手中。
