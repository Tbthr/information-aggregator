## AI快讯

### 产品更新

- [**MiniMax 发布 Music 2.6 音乐模型，新增 Cover 功能**](https://www.minimaxi.com/news/music-26) — MiniMax 发布了 Music 2.6 音乐模型并开启限免。
该版本在底层能力上进行了系统性升级，重点优化了中低频声学表现、指令控制能力以及首包延迟，并新增 Cover 功能。
此外，同步开源了三个 Music Skill，赋能 AI Agent。
- [**Claude Code 推出 ultraplan 支持云端规划本地执行**](https://code.claude.com/docs/en/ultraplan) — Claude Code 推出了名为 ultraplan 的研究预览版新功能，将代码分析与方案起草转移至云端处理，以释放本地资源。
用户在本地终端输入指令发起规划后，可直接在浏览器中对云端生成的方案进行批注和迭代审阅。
确认后，既能在云端执行并创建 PR，也可传回本地继续实施。
- [**Qwen Code 更新，支持远程控制及定时任务**](https://x.com/Alibaba_Qwen/status/2042551216769765449) — Qwen Code 发布更新，开发者现在能通过钉钉或微信等社交平台用手机远程控制执行代码，并可利用定时任务功能让AI自动运行循环测试。
同时，新版支持在执行前进行全局规划，并为不同子任务灵活分配模型。
- [**Anthropic 推出 Claude for Word 插件测试版**](https://claude.com/claude-for-word) — Anthropic 推出了 Claude for Word 插件测试版，该插件能以原生修订模式提供保留原格式的修改建议，还能自动总结修改记录并标记文档错误。目前仅向 Claude 的 Team 和 Enterprise 付费用户开放。
- [**谷歌搜索 AI Mode 餐厅预订功能扩展至全球多地**](https://blog.google/company-news/inside-google/around-the-globe/google-europe/united-kingdom/ai-mode-restaurants-uk/) — 谷歌宣布将其 AI Mode 餐厅预订功能扩展至 澳大利亚、加拿大 和 中国香港 等地。
用户只需在 AI Mode 中用自然语言输入人数、时间等特定偏好，系统就会提供 OpenTable 等八个合作方的预订链接。
- [**Qwen Chat 更名为 Qwen Studio**](https://studio.qwen.ai/) — 阿里 Qwen 宣布其旗下 Qwen Chat 正式更名为 Qwen Studio。未来，其团队将继续优化该平台的界面与体验，并依据用户反馈制定路线图，以帮助用户更便捷地发现、测试并挖掘 Qwen 模型的全部潜力。

### 前沿研究

- [**Anuttacon 发布 LPM 1.0 模型，仅供学术交流**](https://large-performance-model.github.io/) — 由米哈游创始人蔡浩宇创立的AI公司Anuttacon正式公布了自研视频角色表演生成模型LPM 1.0。该模型采用17B参数扩散架构，支持多模态导演级控制与零样本泛化，可作为虚拟主播和游戏NPC等场景的实时视觉引擎。
官方称，该项目现阶段仅用于学术交流，无任何开源计划，也不提供任何API与产品服务。
